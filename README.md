# Estoque Vinhos

Webapp para controle de estoque, localizacao fisica e movimentacoes auditaveis de uma loja de vinhos e espumantes.

## Stack

- Next.js
- React
- TypeScript
- Prisma
- PostgreSQL
- Tailwind CSS

## Primeiros comandos

```bash
npm install
npm run prisma:generate
npm run prisma:migrate
npm run dev
```
